# paulwing-codex-skills

Personal Codex skills maintained by Paul Wing.

## Skills

- `change-planning`: Inspect the current project first, propose a concise implementation plan, and wait for confirmation before making non-trivial codebase changes.

## Install

Install the skill from GitHub with Codex's skill installer:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py --repo paulwing/paulwing-codex-skills --path skills/change-planning
```

Restart Codex after installing so it can discover the new skill.
